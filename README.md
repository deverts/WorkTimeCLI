# WorkTimeCLI (WIP)

WorkTimeCLI--in it's current form--helps add time tracking to Asana. This is helpful as a standalone feature, but is very helpful for teams when they're looking to make estimates in future. Check the roadmap for more info to help colour this.

This project is still in progress. It's stable, but has not been refactored, and is currently missing install scripts.

WorkTimeCLI has a couple of purposes:
- An interface to help run different experiments with task trackers
- A playground to experiment with different packages, different techniques etc

This, generally, is not supposed to be a full replacement to the Asana web UI. It's a tool to speed up common actions, and provide useful extra actions that the web interface does not support.
