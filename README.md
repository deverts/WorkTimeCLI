# WorkTimeCLI

WorkTimeCLI has a couple of purposes:
- An interface to help run different experiments with task trackers
- A playground to experiment with different packages, different techniques etc

This, generally, is not supposed to be a full replacement to the Asana web UI. Look at it
as a tool to speed up common actions, and provide useful extra actions that the web interface does not support.


## Supported platforms:
* Asana
